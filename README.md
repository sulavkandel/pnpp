# Pokhara Mahanagarpalika Citizen Portal

Project structure for a citizen complaint portal with separate citizen, ward, department, and admin experiences.

## Structure

- `index.html`: current static entry for the prototype
- `frontend/`: frontend assets and source files
- `backend/`: backend-ready Node/Express structure

## Backend next step

Use server-side environment variables for MongoDB credentials. Do not expose database secrets in frontend files.
